# Data Cleaning using Pandas

## Objective
Perform basic data exploration and cleaning using Pandas.

## Dataset
The dataset contains 1000 product records with 24 attributes including product details, ratings, prices, discounts, and categories.

## Tasks Completed
- Loaded dataset into Pandas DataFrame
- Explored data structure and statistics
- Handled missing values
- Performed filtering and column selection
- Checked for duplicates
- Created derived column `discount_amount`
- Saved cleaned dataset as CSV
