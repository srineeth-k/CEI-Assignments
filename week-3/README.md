# Week 3 Data Engineering Assignment

## Summary

In this assignment, I worked with the Superstore Sales dataset using MySQL and applied advanced SQL concepts such as Subqueries, Common Table Expressions (CTEs), Window Functions, and Joins. The goal was to analyze customer sales performance and generate business insights from sales transaction data.

The dataset was first imported into a staging table named `superstore_raw`. From this table, separate `customers`, `products`, and `orders` tables were created for analysis. Various SQL queries were then executed to identify top-performing customers, above-average sales, customer rankings, and order-level insights.

---

## Import and Setup Data

The Superstore dataset was imported into MySQL Workbench and stored in a table named `superstore_raw`.

To organize the data, three separate tables were created:

* `customers`
* `products`
* `orders`

Data was inserted into these tables using `SELECT DISTINCT` wherever required to avoid duplicate records.

---

## Subquery Analysis

### Orders with Sales Greater than Average

A subquery was used to identify orders whose sales value was greater than the overall average sales value.

### Highest Sales Order per Customer

A subquery was used to find the highest-value order placed by each customer.

---

## CTE Analysis

### Total Sales per Customer

A Common Table Expression (CTE) was used to calculate the total sales generated by each customer.

### Customers with Above-Average Sales

The customer sales CTE was combined with a subquery to identify customers whose total sales were higher than the average customer sales.

---

## Window Function Analysis

### Customer Ranking

The `RANK()` function was used to rank customers based on total sales.

### Order Numbering

The `ROW_NUMBER()` function with `PARTITION BY` was used to assign row numbers to orders within each customer group.

### Top 3 Customers

Window functions were used to identify the top 3 customers based on total sales.

---

## Final Combined Query

A final query was created using:

* JOIN
* CTE
* Window Function

This query displayed:

* Customer Name
* Total Sales
* Sales Rank

This provided a complete customer sales ranking report.
